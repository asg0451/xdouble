# xdouble

Live video translation app for macOS. Captures another app's window (like iPhone Mirroring) and displays a translated version with Chinese text replaced by English.

## Requirements
- macOS 15+ (Sequoia)
- Apple Silicon Mac
- Screen Recording permission
